# Amazon Clone

## Description
This is a **responsive Amazon e-commerce clone** built using **HTML** and **CSS**.  
It replicates the basic structure and layout of Amazon's homepage, including the header, footer, product sections, and responsive design for both desktop and mobile screens.

---

## Features
- **Header**: Amazon logo, search bar, navigation links.  
- **Footer**: Language selector with India flag, links for policies, and info.  
- **Product Sectio**n: Grid layout showcasing products with images.  
- **Responsive Design**: Works well on desktop, tablet, and mobile screens.  
- **Assets**: Includes all images used for product sections and logos.
